      • getListData

      getListData(set)

      Retrieves list data for a particular set given the url settings. If url.getListData is a function, that function will be called. If url.getListData is a string, a request to that string will be made. If url is a string, a GET request is made to url.

      Parameters

      1. set {Set}:

        A object that represents the set of data needed to be loaded.

      Returns

      {Promise<ListData>}:

      A promise that resolves to the ListData format.
