      • serializeList

      Returns the serialized form of an list.

      connection.serializeList( list )

      This implementation simply returns an Array containing the result of serializeInstance called on each item in the list.

      Parameters

      1. list {List}:

        The instance to serialize.

      Returns

      {Object|Array}:

      A serialized representation of the list.
