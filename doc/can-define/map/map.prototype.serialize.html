      • serialize

      Get a serialized representation of the map instance and its children.

      map.serialize()

      Get the serialized Object form of the map. Serialized data is typically used to send back to a server. Use serialize to customize a property's serialized value or if the property should be added to the result or not.

      undefined serialized values are not added to the result.

      var MyMap = DefineMap.extend({
  date: {
    type: "date",
    serialize: function(date){
      return date.getTime()
    }
  }
});

var myMap = new MyMap({date: new Date(), count: 5});
myMap.serialize() //-> {date: 1469566698504, count: 5}

      Returns

      {Object}:

      A JavaScript Object that can be serialized with JSON.stringify or other methods.
