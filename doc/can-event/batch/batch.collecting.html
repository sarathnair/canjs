      batch.collecting()

      Returns the Queue that is currently collecting tasks.

      batch.start();
batch.collecting() //-> Queue

batch.stop();
batch.collecting() //-> null

      Returns

      {can-event/batch/Queue}:

      The queue currently collecting tasks.
