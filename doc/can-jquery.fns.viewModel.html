      • viewModel

      Gets an element's View Model.

      .viewModel()

      Calls can-view-model with the unwrapped HTMLElement.

      var vm = $("my-component").viewModel();

      Returns

      {can-define/map/map|can-map|Object}:

      Returns the View Model set for this element.
